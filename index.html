The Art of Compression

Using a similar concept to Quadtrees, created an algorithm that recreates classic images with a blocky pixelated effect. The algorithmn stores the colour values of the pixels and creates a binary tree to store the information held in those pixels. Every node contains a pixel that represents the average color over the rectangle in the original image. When a rectangle is split into two smaller rectangles, the parent node contains the whole rectangle, the left child contains the parent’s upper left corner and a new lower right corner, and the right child contains a new upper left corner and the parent’s lower right corner. Rectangles can be split either horizontally or vertically. Splitting is performed to minimize color variability across the resulting rectangles. Written in C++.

  • 28 March 2018
  • https://github.com/Cameron-lees
